# Synx + JS Framework Benchmark (JSX)

Separate TSX/JSX variant of the local benchmark implementation.

## Run locally

```bash
pnpm examples:js-framework-benchmark:jsx
```

This keeps the same benchmark DOM contract (`#run`, `#runlots`, `#add`, `#update`, `#clear`, `#swaprows`, row select/remove classes) while using `@synx/jsx` for rendering.
